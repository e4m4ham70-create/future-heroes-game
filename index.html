Mission Rules

  1. 총 10개의 문제를 순서대로 풉니다.
  2. 각 문제는 총 3번의 기회가 있습니다.
  3. 정답을 맞히면 ACCESS GRANTED 효과가 나타납니다.
  4. 오답을 입력하면 ACCESS DENIED 효과가 나타납니다.
  5. 1~9번 문제의 정답을 맞힐 때마다 inclusion의 철자 중 하나를 얻습니다.
  6. 문제를 3번 안에 맞히지 못하면 해당 Secret Code 철자를 얻을 수 없습니다.
  7. 알아낸 Secret Code 타일은 다음 문제에서도 계속 보입니다.
  8. 모든 문제를 푼 후 최종 컴퓨터 화면에 Secret Code를 입력합니다.
  9. Secret Code를 맞히면 Digital World를 구할 수 있습니다.
